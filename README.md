# 足迹地图

记录你走过的每一个城市，导出为图片分享。

## 使用

直接浏览器打开 `index.html` 即可。

- **搜索** — 搜索城市名，点击添加
- **地图点击** — 点击城市/省份切换访问状态
- **省/市切换** — 侧边栏切换省级和市级视图
- **配色切换** — 侧边栏切换地图配色方案
- **导出图片** — 导出当前地图为 PNG
- **快捷键** — `/` 搜索、`Ctrl+S` 导出

## 技术栈

- **地图**: Mapbox GL JS v3.3.0
- **数据**: 中国省/市 GeoJSON
- **存储**: localStorage
- **字体**: Noto Serif SC / Noto Sans SC
